# ðŸ”„ Integral (Antiturunan)

**Integral** secara umum dikenal sebagai kebalikan (invers) dari turunan. Jika turunan menurunkan pangkat, maka integral "menaikkan" pangkat.  Integral memiliki banyak aplikasi penting, seperti menghitung luas daerah di bawah kurva.

---

## 1. Pengertian dan Jenis Integral

### 1.1 Integral Tak Tentu

Integral tak tentu adalah operasi kebalikan dari turunan, menghasilkan fungsi dengan konstanta integrasi <Latex>{`C`}</Latex>.

<Latex block>{`\\int f(x) \\, dx = F(x) + C`}</Latex>

<Callout type="info" emoji="â“">
  **Mengapa ada konstanta C?** Karena turunan dari konstanta adalah 0, maka saat kita "membalik" proses turunan, kita tidak tahu konstanta aslinya. Contoh: turunan dari <Latex>{`x^2 + 5`}</Latex> dan <Latex>{`x^2 + 100`}</Latex> sama-sama <Latex>{`2x`}</Latex>.
</Callout>

### 1.2 Integral Tentu

Integral tentu menghitung nilai numerik (luas) dengan batas atas dan batas bawah tertentu.

<Latex block>{`\\int_{a}^{b} f(x) \\, dx = F(b) - F(a)`}</Latex>

Dimana:
- <Latex>{`a`}</Latex> = batas bawah
- <Latex>{`b`}</Latex> = batas atas
- <Latex>{`F(x)`}</Latex> = antiturunan dari <Latex>{`f(x)`}</Latex>

---

## 2. Rumus Dasar

### 2.1 Aturan Pangkat (Power Rule)

<Latex block>{`\\int ax^n \\, dx = a \\cdot \\frac{1}{n+1}x^{n+1} + C \\quad \\text{(untuk } n \\neq -1 \\text{)}`}</Latex>

<Callout type="info" emoji="ðŸ’¡">
  **Cara Cepat:** "Pangkat ditambah satu, lalu koefisien dibagi dengan pangkat yang baru."
</Callout>

### 2.2 Tabel Rumus Penting

| Fungsi | Integral |
|--------|----------|
| <Latex>{`\\int k \\, dx`}</Latex> | <Latex>{`kx + C`}</Latex> |
| <Latex>{`\\int x^n \\, dx`}</Latex> | <Latex>{`\\frac{1}{n+1}x^{n+1} + C`}</Latex> |
| <Latex>{`\\int \\frac{1}{x} \\, dx`}</Latex> | <Latex>{`\\ln\|x\| + C`}</Latex> |
| <Latex>{`\\int (f \\pm g) \\, dx`}</Latex> | <Latex>{`\\int f \\, dx \\pm \\int g \\, dx`}</Latex> |

## 3. Contoh Soal Integral Tak Tentu

### Soal 1: Integral Konstanta

**Soal:** Tentukan hasil dari <Latex>{`\\int 4 \\, dx`}</Latex>.

**Penyelesaian:**

Anggap <Latex>{`4 = 4x^0`}</Latex>, lalu gunakan aturan pangkat:

<Latex block>{`
\\begin{aligned}
\\int 4 \\, dx &= \\int 4x^0 \\, dx \\\\[8pt]
&= 4 \\cdot \\frac{x^{0+1}}{0+1} + C \\\\[8pt]
&= 4 \\cdot \\frac{1}{1}x^1 + C \\\\[8pt]
&= 4x + C
\\end{aligned}
`}</Latex>

---

### Soal 2: Integral Bentuk Pecahan

**Soal:** Tentukan hasil dari <Latex>{`\\int \\frac{2}{x^2} \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Ubah ke bentuk pangkat negatif

<Latex block>{`\\frac{2}{x^2} = 2x^{-2}`}</Latex>

**Langkah 2:** Integralkan

<Latex block>{`
\\begin{aligned}
\\int 2x^{-2} \\, dx &= 2 \\cdot \\frac{x^{-2+1}}{-2+1} + C \\\\[8pt]
&= 2 \\cdot \\frac{1}{-1}x^{-1} + C \\\\[8pt]
&= -2x^{-1} + C \\\\[8pt]
&= -\\frac{2}{x} + C
\\end{aligned}
`}</Latex>

---

### Soal 3: Perkalian Polinomial (Selisih Kuadrat)

**Soal:** Tentukan hasil dari <Latex>{`\\int (x-3)(x+3) \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Gunakan rumus selisih kuadrat <Latex>{`(a-b)(a+b) = a^2 - b^2`}</Latex>

<Latex block>{`(x-3)(x+3) = x^2 - 9`}</Latex>

**Langkah 2:** Integralkan

<Latex block>{`
\\begin{aligned}
\\int (x^2 - 9) \\, dx &= \\frac{1}{2+1}x^{2+1} - 9x + C \\\\[8pt]
&= \\frac{x^3}{3} - 9x + C
\\end{aligned}
`}</Latex>

---

### Soal 4: Polinomial Campuran

**Soal:** Tentukan hasil dari <Latex>{`\\int (3x^2 - 4x + 5) \\, dx`}</Latex>.

**Penyelesaian:**

Integralkan setiap suku secara terpisah: 

| Suku | Proses | Hasil |
|------|--------|-------|
| <Latex>{`3x^2`}</Latex> | <Latex>{`3 \\cdot \\frac{1}{3}x^3`}</Latex> | <Latex>{`x^3`}</Latex> |
| <Latex>{`-4x`}</Latex> | <Latex>{`-4 \\cdot \\frac{1}{2}x^2`}</Latex> | <Latex>{`-2x^2`}</Latex> |
| <Latex>{`5`}</Latex> | <Latex>{`5x`}</Latex> | <Latex>{`5x`}</Latex> |

**Hasil Akhir:** <Latex>{`x^3 - 2x^2 + 5x + C`}</Latex>

---

### Soal 5: Perkalian Polinomial Lanjut

**Soal:** Tentukan hasil dari <Latex>{`\\int (4x^2 - 3x)(5x + 7) \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Kalikan kedua kurung (distribusi)

<Latex block>{`
\\begin{aligned}
(4x^2 - 3x)(5x + 7) &= 4x^2(5x) + 4x^2(7) - 3x(5x) - 3x(7) \\\\[8pt]
&= 20x^3 + 28x^2 - 15x^2 - 21x \\\\[8pt]
&= 20x^3 + 13x^2 - 21x
\\end{aligned}
`}</Latex>

**Langkah 2:** Integralkan setiap suku

<Latex block>{`
\\begin{aligned}
\\int (20x^3 + 13x^2 - 21x) \\, dx &= 20 \\cdot \\frac{1}{4}x^4 + 13 \\cdot \\frac{1}{3}x^3 - 21 \\cdot \\frac{1}{2}x^2 + C \\\\[8pt]
&= 5x^4 + \\frac{13}{3}x^3 - \\frac{21}{2}x^2 + C
\\end{aligned}
`}</Latex>

---

### Soal 6: Pangkat Pecahan

**Soal:** Tentukan hasil dari <Latex>{`\\int \\frac{3}{7}x^{6/9} \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Sederhanakan pangkat pecahan

<Latex block>{`\\frac{6}{9} = \\frac{2}{3}`}</Latex>

**Langkah 2:** Hitung <Latex>{`n + 1`}</Latex>

<Latex block>{`n + 1 = \\frac{2}{3} + 1 = \\frac{2}{3} + \\frac{3}{3} = \\frac{5}{3}`}</Latex>

**Langkah 3:** Integralkan

<Latex block>{`
\\begin{aligned}
\\int \\frac{3}{7}x^{2/3} \\, dx &= \\frac{3}{7} \\cdot \\frac{1}{5/3}x^{5/3} + C \\\\[8pt]
&= \\frac{3}{7} \\cdot \\frac{3}{5} x^{5/3} + C \\\\[8pt]
&= \\frac{9}{35} x^{5/3} + C
\\end{aligned}
`}</Latex>

**Bentuk Akar:** <Latex>{`\\frac{9}{35} \\sqrt[3]{x^5} + C`}</Latex>

---

### Soal 7: Pembagian Variabel (Pangkat Negatif)

**Soal:** Tentukan hasil dari <Latex>{`\\int \\frac{7x^3}{6x^{10}} \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Sederhanakan menggunakan aturan eksponen <Latex>{`\\frac{x^a}{x^b} = x^{a-b}`}</Latex>

<Latex block>{`\\frac{7x^3}{6x^{10}} = \\frac{7}{6} x^{3-10} = \\frac{7}{6} x^{-7}`}</Latex>

**Langkah 2:** Integralkan dengan <Latex>{`n + 1 = -7 + 1 = -6`}</Latex>

<Latex block>{`
\\begin{aligned}
\\int \\frac{7}{6} x^{-7} \\, dx &= \\frac{7}{6} \\cdot \\frac{1}{-6}x^{-6} + C \\\\[8pt]
&= -\\frac{7}{36} x^{-6} + C \\\\[8pt]
&= -\\frac{7}{36x^6} + C
\\end{aligned}
`}</Latex>

---

### Soal 8: Perkalian dan Pembagian Polinomial

**Soal:** Tentukan hasil dari <Latex>{`\\int \\frac{(3x^2 - 8)(3x^2 + 8)}{12x^4} \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Jabarkan pembilang dengan selisih kuadrat

<Latex block>{`
\\begin{aligned}
(3x^2 - 8)(3x^2 + 8) &= (3x^2)^2 - 8^2 \\\\[8pt]
&= 9x^4 - 64
\\end{aligned}
`}</Latex>

**Langkah 2:** Bagi setiap suku dengan penyebut

<Latex block>{`
\\begin{aligned}
\\frac{9x^4 - 64}{12x^4} &= \\frac{9x^4}{12x^4} - \\frac{64}{12x^4} \\\\[8pt]
&= \\frac{3}{4} - \\frac{16}{3}x^{-4}
\\end{aligned}
`}</Latex>

**Langkah 3:** Integralkan setiap suku

<Latex block>{`
\\begin{aligned}
\\int \\left( \\frac{3}{4} - \\frac{16}{3}x^{-4} \\right) dx &= \\frac{3}{4}x - \\frac{16}{3} \\cdot \\frac{1}{-3}x^{-3} + C \\\\[8pt]
&= \\frac{3}{4}x + \\frac{16}{9}x^{-3} + C \\\\[8pt]
&= \\frac{3}{4}x + \\frac{16}{9x^3} + C
\\end{aligned}
`}</Latex>

## 4. Integral Tentu

### 4.1 Konsep Dasar

Integral tentu memiliki batas atas dan batas bawah, sehingga menghasilkan nilai numerik (bukan fungsi).

<Latex block>{`\\int_{a}^{b} f(x) \\, dx = [F(x)]_{a}^{b} = F(b) - F(a)`}</Latex>

<Callout type="warning" emoji="âš ï¸">
  **Perhatian!** Pada integral tentu, konstanta <Latex>{`C`}</Latex> tidak perlu ditulis karena akan saling menghilangkan saat substitusi batas.
</Callout>

### 4.2 Contoh Soal

**Soal:** Hitung nilai dari <Latex>{`\\int_{1}^{2} 4x^3 \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Integralkan fungsi

<Latex block>{`\\int 4x^3 \\, dx = 4 \\cdot \\frac{1}{4}x^4 = x^4`}</Latex>

**Langkah 2:** Substitusi batas atas dan batas bawah

<Latex block>{`
\\begin{aligned}
[x^4]_{1}^{2} &= (2)^4 - (1)^4 \\\\[8pt]
&= 16 - 1 \\\\[8pt]
&= 15
\\end{aligned}
`}</Latex>

## 5. Teknik Substitusi

### 5.1 Kapan Menggunakan Substitusi? 

Teknik substitusi digunakan jika terdapat **fungsi di dalam fungsi** (komposisi) yang tidak bisa langsung diintegralkan dengan aturan pangkat biasa.

### 5.2 Langkah-Langkah Substitusi

1.  Identifikasi bagian dalam fungsi, misalkan sebagai <Latex>{`u`}</Latex>
2. Hitung <Latex>{`\\frac{du}{dx}`}</Latex>, lalu nyatakan <Latex>{`dx`}</Latex> dalam <Latex>{`du`}</Latex>
3. Substitusi semua variabel <Latex>{`x`}</Latex> menjadi <Latex>{`u`}</Latex>
4. Integralkan dalam variabel <Latex>{`u`}</Latex>
5. Kembalikan ke variabel <Latex>{`x`}</Latex>

### 5.3 Contoh Soal

**Soal:** Tentukan hasil dari <Latex>{`\\int (2x+3)^5 \\, dx`}</Latex>.

**Penyelesaian:**

**Langkah 1:** Misalkan fungsi dalam

<Latex block>{`u = 2x + 3`}</Latex>

**Langkah 2:** Turunkan untuk mendapatkan <Latex>{`du`}</Latex>

<Latex block>{`\\frac{du}{dx} = 2 \\quad \\Rightarrow \\quad dx = \\frac{du}{2}`}</Latex>

**Langkah 3:** Substitusi ke integral

<Latex block>{`\\int (2x+3)^5 \\, dx = \\int u^5 \\cdot \\frac{du}{2} = \\frac{1}{2} \\int u^5 \\, du`}</Latex>

**Langkah 4:** Integralkan

<Latex block>{`
\\begin{aligned}
\\frac{1}{2} \\int u^5 \\, du &= \\frac{1}{2} \\cdot \\frac{1}{6}u^6 + C \\\\[8pt]
&= \\frac{u^6}{12} + C
\\end{aligned}
`}</Latex>

**Langkah 5:** Kembalikan ke variabel <Latex>{`x`}</Latex>

<Latex block>{`= \\frac{(2x+3)^6}{12} + C`}</Latex>

## 6. Ringkasan

### 6.1 Perbandingan Turunan dan Integral

| Turunan | Integral |
|---------|----------|
| <Latex>{`x^n \\to nx^{n-1}`}</Latex> | <Latex>{`x^n \\to \\frac{1}{n+1}x^{n+1}`}</Latex> |
| Pangkat turun, dikali pangkat lama | Pangkat naik, dibagi pangkat baru |
| Tidak ada konstanta | Ada konstanta <Latex>{`C`}</Latex> |

### 6.2 Checklist Sebelum Mengintegralkan

âœ… Sederhanakan fungsi terlebih dahulu (jabarkan perkalian, bagi pecahan)

âœ… Ubah akar dan pecahan ke bentuk pangkat

âœ… Periksa apakah perlu menggunakan teknik substitusi

âœ… Jangan lupa konstanta <Latex>{`C`}</Latex> untuk integral tak tentu

## ðŸ’¡ Tips Mengerjakan Soal

> **Sederhanakan dulu!** Selalu jabarkan perkalian polinomial dan sederhanakan pecahan sebelum mengintegralkan. 

> **Ubah ke bentuk pangkat!** Akar (<Latex>{`\\sqrt{x} = x^{1/2}`}</Latex>) dan pecahan (<Latex>{`\\frac{1}{x^2} = x^{-2}`}</Latex>) harus diubah ke bentuk pangkat.

> **Hati-hati dengan tanda negatif!** Terutama saat pangkat negatif ditambah 1, hasilnya tetap negatif (misal:  <Latex>{`-7 + 1 = -6`}</Latex>).

> **Periksa jawaban dengan menurunkan!** Cara terbaik memverifikasi integral adalah dengan menurunkan hasilnyaâ€”harus kembali ke fungsi awal.
