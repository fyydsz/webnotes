# ðŸ“‰ Turunan (Derivatif)

**Turunan** adalah pengukuran terhadap bagaimana nilai suatu fungsi berubah seiring perubahan nilai inputnya. Secara geometris, turunan merepresentasikan **kemiringan (gradien)** garis singgung pada suatu titik di kurva fungsi.

## 1. Konsep Dasar (Definisi Limit)

Secara formal, turunan didefinisikan menggunakan limit. Kita menggunakan notasi <Latex>{`\\Delta x`}</Latex> untuk melambangkan perubahan kecil pada variabel <Latex>{`x`}</Latex>.

<Latex block>{`
  f'(x) = \\lim_{\\Delta x \\to 0} \\frac{f(x + \\Delta x) - f(x)}{\\Delta x}
`}</Latex>

---

## 2. Rumus-Rumus Dasar

Untuk mempercepat perhitungan, kita menggunakan aturan praktis berikut:

### A. Turunan Konstanta
Jika <Latex>{`f(x) = k`}</Latex> (angka saja), maka turunannya selalu nol.
<Latex block>{`f'(x) = 0`}</Latex>

### B. Aturan Pangkat (Power Rule)
Jika <Latex>{`f(x) = ax^n`}</Latex>, maka:

<Latex block>{`f'(x) = n \\cdot a x^{n-1}`}</Latex>

<Callout type="info" emoji="ðŸ’¡">
  **Cara Cepat:** "Pangkat turun ke depan dikali koefisien, lalu pangkatnya dikurangi satu."
</Callout>

### C. Penjumlahan & Pengurangan
Turunan dari penjumlahan fungsi adalah jumlah dari turunannya.
<Latex block>{`y' = a'(x) \\pm b'(x)`}</Latex>

---

## 3. Contoh Soal Dasar

### Soal 1: Polinomial
Tentukan turunan dari <Latex>{`y = 2x^4 - 6x + 15`}</Latex>.

**Penyelesaian:**
1. <Latex>{`2x^4 \\to 4 \\cdot 2x^3 = 8x^3`}</Latex>
2. <Latex>{`-6x \\to 1 \\cdot -6x^0 = -6`}</Latex>
3. <Latex>{`15 \\to 0`}</Latex>

Hasil akhir: <Latex>{`y' = 8x^3 - 6`}</Latex>

### Soal 2: Akar & Pecahan
Tentukan turunan dari <Latex>{`f(x) = \\sqrt{x} + \\frac{1}{x^2}`}</Latex>.

**Penyelesaian:**
Ubah dulu ke bentuk pangkat: <Latex>{`f(x) = x^{1/2} + x^{-2}`}</Latex>.

<Latex block>{`
  \\begin{aligned}
  f'(x) &= \\frac{1}{2}x^{-1/2} - 2x^{-3} \\\\
  &= \\frac{1}{2\\sqrt{x}} - \\frac{2}{x^3}
  \\end{aligned}
`}</Latex>

---

## 4. Aturan Lanjut (Advanced Rules)

### D. Aturan Rantai (Chain Rule)
Digunakan untuk fungsi komposisi (fungsi di dalam fungsi), seperti <Latex>{`y = (u(x))^n`}</Latex>.

<Latex block>{`
  f'(x) = n \\cdot (u(x))^{n-1} \\cdot u'(x)
`}</Latex>

<Callout type="info" emoji="ðŸ”—">
  **Konsep:** Turunkan fungsi luarnya (pangkat), lalu **kalikan** dengan turunan fungsi dalamnya.
</Callout>

**Contoh:** Tentukan turunan <Latex>{`y = (3x^2 - 5)^4`}</Latex>.
* Fungsi dalam: <Latex>{`u = 3x^2 - 5 \\to u' = 6x`}</Latex>
* Fungsi luar: <Latex>{`(\\dots)^4`}</Latex>

<Latex block>{`
  \\begin{aligned}
  y' &= 4(3x^2 - 5)^{4-1} \\cdot (6x) \\\\
  &= 24x(3x^2 - 5)^3
  \\end{aligned}
`}</Latex>


### E. Aturan Perkalian (Product Rule)
Digunakan jika <Latex>{`f(x)`}</Latex> adalah hasil kali dua fungsi, kita sebut saja fungsi **a** (depan) dan fungsi **b** (belakang).

<Latex block>{`
  \\text{Jika } f(x) = a \\cdot b \\quad \\text{maka} \\quad f'(x) = a' \\cdot b + a \\cdot b'
`}</Latex>

<Callout type="info" emoji="ðŸ’¡">
  **Hafalan:** "Turunan depan ($a'$) kali belakang ($b$), ditambah depan ($a$) kali turunan belakang ($b'$)."
</Callout>

**Contoh:** Tentukan turunan <Latex>{`y = (2x + 3)(x^2)`}</Latex>.
* Misal <Latex>{`a = 2x + 3 \\to a' = 2`}</Latex>
* Misal <Latex>{`b = x^2 \\to b' = 2x`}</Latex>

<Latex block>{`
  \\begin{aligned}
  y' &= (2)(x^2) + (2x+3)(2x) \\\\
  &= 2x^2 + 4x^2 + 6x \\\\
  &= 6x^2 + 6x
  \\end{aligned}
`}</Latex>

### F. Aturan Pembagian (Quotient Rule)
Digunakan jika <Latex>{`f(x)`}</Latex> adalah pembagian dua fungsi, kita sebut fungsi **a** (atas) dan fungsi **b** (bawah).

<Latex block>{`
  \\text{Jika } f(x) = \\frac{a}{b} \\quad \\text{maka} \\quad f'(x) = \\frac{a' \\cdot b - a \\cdot b'}{b^2}
`}</Latex>

<Callout type="warning" emoji="âš ï¸">
  **Hati-hati!** Tandanya adalah **kurang (-)**, dan urutannya tidak boleh terbalik.
  **Hafalan:** "Turunan atas ($a'$) kali bawah ($b$), **dikurang** atas ($a$) kali turunan bawah ($b'$), dibagi bawah kuadrat ($b^2$)."
</Callout>

**Contoh:** Tentukan turunan <Latex>{`y = \\frac{x^2}{x+1}`}</Latex>.
* Atas: <Latex>{`a = x^2 \\to a' = 2x`}</Latex>
* Bawah: <Latex>{`b = x+1 \\to b' = 1`}</Latex>

<Latex block>{`
  \\begin{aligned}
  y' &= \\frac{(2x)(x+1) - (x^2)(1)}{(x+1)^2} \\\\
  &= \\frac{2x^2 + 2x - x^2}{(x+1)^2} \\\\
  &= \\frac{x^2 + 2x}{(x+1)^2}
  \\end{aligned}
`}</Latex>
